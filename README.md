## **Short Description**

**PageTurner** is a full-stack online bookstore developed using the **MERN stack**. It allows users to **browse, search, and purchase books online**, while providing an **admin dashboard** to manage books, orders, and users. The system replaces the traditional, manual bookstore process with a **fast, efficient, and user-friendly digital experience**.

The platform includes features like **authentication, book catalogue management, shopping cart, order tracking, and role-based access** for admin and customers.
